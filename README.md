Este projeto implementa um pipeline MLOps para fine-tuning de modelos de linguagem (LLMs) usando:

- **Compute**: Google Colab (GPU T4 gratuita)
- **Orquestra├Д├Бo**: GitHub Actions
- **Tracking**: DagsHub + MLflow
- **Modelos**: Llama 3.2 (1B/3B) ou Qwen
- **Otimiza├Д├Бo**: QLoRA + bitsandbytes (4-bit quantization)

## Quick Start

### 1. Configurar DagsHub

1. Crie uma conta em [DagsHub](https://dagshub.com)
2. Crie um novo reposit├│rio
3. Copie as credenciais (Settings Рєњ Access Tokens)

### 2. Configurar Secrets no GitHub

Adicione os seguintes secrets no seu reposit├│rio:

- `DAGSHUB_TOKEN`: Seu token do DagsHub
- `DAGSHUB_USERNAME`: Seu username do DagsHub

### 3. Executar no Google Colab

1. Abra o notebook `notebooks/fine_tuning_pipeline.ipynb` no Colab
2. Conecte a uma GPU (Runtime Рєњ Change runtime type Рєњ T4 GPU)
3. Configure suas credenciais nas vari├Аveis de ambiente
4. Execute todas as c├Еlulas

### 4. Trigger via GitHub Actions

```bash
# Push de novo dataset dispara o workflow
git add data/
git commit -m "feat: novo dataset para treinamento"
git push
```
